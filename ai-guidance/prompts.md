# AI Prompt Engineering Strategy

To ensure Refract delivers professional-grade analysis, I utilized structured prompting techniques.

## Backend Analysis Prompt

- **Model:** GPT-4o
- **Strategy:** Structured Output Generation
- **Prompt:** "Act as a Senior UX Researcher. Analyze the provided transcript. Extract:
  1. A sentiment score (Positive, Neutral, Negative).
  2. A list of 'Pain Points' (Summary + Verbatim Quote).
  3. A list of 'Feature Requests' (Summary + Verbatim Quote).
     Output MUST be strictly valid JSON. Do not include any conversational text or markdown blocks."

## Frontend UI Prompt

- **Strategy:** Utility-First Design
- **Prompt:** "Generate React components using Tailwind CSS. Focus on a high-end SaaS dashboard aesthetic. Use a clean color palette (Indigo/Slate) and Lucide-React for all icons. Ensure the layout is responsive and emphasizes scannability."
